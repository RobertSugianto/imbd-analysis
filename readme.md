# Foreword
This project was my personal project on analyzing and IMDB data. I use python notebook for data cleaning and tableau for data visualization. I do descriptive data analysis to understand the correlation between gross and each other attributes. The objective of this project is to understand which director has the most gross and which genre combination is most chosen.
# Dataset
The dataset was taken from kaggle in this link \
https://www.kaggle.com/datasets/harshitshankhdhar/imdb-dataset-of-top-1000-movies-and-tv-shows

# Requirement
to run the python notebook file, you must have pandas install in your python library.
